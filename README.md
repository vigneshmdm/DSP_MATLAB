# DSP Fundamentals in MATLAB

This repository contains various **Digital Signal Processing (DSP)** algorithms and implementations in **MATLAB**. The code includes **FIR** and **IIR** filter designs, **Fourier transforms**, **convolution operations**, and more.

## Contents
- **Filter Design**:
  - Butterworth Filters: Low-pass, High-pass, Band-pass, Band-stop
  - Chebyshev Filters: Low-pass, High-pass, Band-pass, Band-stop
  - FIR Filters: Low-pass, High-pass, Band-pass, Band-stop
  
- **Signal Processing**:
  - DFT/IDFT (with and without inbuilt functions)
  - Circular and Linear Convolution (with and without inbuilt functions)
  - Spectral Analysis
  - Unit Impulse, Unit Ramp, Unit Step signals

- **Miscellaneous**:
  - Autocorrelation and Cross-correlation examples
